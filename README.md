# Personal Library Dashboard

A full-stack web application for managing personal book collections, tracking reading progress, and visualising my/your reading habits.

## Tech Stack

### Frontend
- React
- TypeScript

### Backend
- FastAPI
- SQLAlchemy

### Database
- PostgreSQL

## Features
- Book management
- Reading progress tracking
- Search and filtering
- Reading analytics
- AI-powered recommendations (planned)
